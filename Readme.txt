הנחייה ראשונה:  תקיית "resources", קובץ ה-JAR וקובץ ה-bat *חייבים* להיות כולם באותה תקייה
הנחייה שנייה: הרצה מקובץ ה-bat בלבד.
הנחייה שלישית: הררכיית קבצי הקורפוס: בהינתן נתיב X לכפתור "corpus path" ב-GUI
קבצי הקורפוס חייבים להיות ב: X\corpus\corpusDirectories. 
בנוסף לכך, קובץ ה-stop_words ייקרא כמו שמופיע במודל (עם 05 בהתחלה ואז רווח ואז stop_words) ויישמר בנתיב X\05 stop_words.txt

הסבר על ה-GUI:
כפתור "Select Corpus Path" - נועד לבחירת הנתיב אל הקורפוס (לפי ההנחייה למעלה)
כפתור "Select Posting Path" - נועד לבחירת הנתיב בו יישמרו כל קבצי הפוסטיג (Terms, Documents) והמילון.
כפתור "Stemming" - נועד לאפשר את האופציה לאינדוקס עם Stemming (כאשר יש וי) או בלי Stemming (כאשר הריבוע ריק)
כפתור "Reset" - נועד למחוק את כל התוכן בנתיב שהוגדר להיות Posting Path. *התקייה עצמה שהמשתמש נתן לא תמחק אלא רק התוכן בפנים*
כפתור "Show Dictionary" - נועד להציג בצורה ממויינת את כל ה-Terms במילון ההופכי ומספר הופעתם בכל הקורפוס בהתאם לסימון בכפתור "Stemming". 
כפתור "Load Dictionary" - נועד להעלות לזיכרון הראשי את המילון ההופכי בהתאם למסומן בכפתור "Stemming" מנתיב שהושם בכפתור ה-"Posting Path".
כפתור "Start" - נועד להתחיל את תהליך האינדוקס כאשר: הושם נתיב לקבצי ה-Corpus (כפי שהונחה למעלה) והושם נתיב לקבצי ה-Posting.

הנחייה רביעית: לאחר שכפתור ה-"Start" נלחץ ותהליך האינדוקס התחיל, אין לגעת בכלל בחלון ה-GUI.
